# Day 10 - 45. Jump Game II

## 🧩 Problem
Find the minimum number of jumps to reach the last index of the array.

## 💡 Approach
- Greedy approach: Maintain current max reach and the reach of the next jump.
- When we reach the end of the current jump, increment jump count and update to next reach.

## ⏱️ Complexity
- Time: O(n)
- Space: O(1)
