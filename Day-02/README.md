# Day 02 - 27. Remove Element

## 🧩 Problem
Remove all occurrences of a value from an array in-place and return the new length.

## 💡 Approach
- Use two pointers: one to traverse, one to track non-val elements
- Shift non-val elements to the front of the array

## ⏱️ Complexity
- Time: O(n)
- Space: O(1)
